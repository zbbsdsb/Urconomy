Innovation = a change that can be 'directly seen'? 
Progress = a perceptible leap? 
The world = something that can be understood intuitively? 
Fruits grow higher 
Innovation is harder 
More tools are needed 
But that’s not the problem!!! 
The problem is—— 
We are still using 'picking' to understand innovation: changes in life visible to the naked eye; 
'Picking' means obtaining directly 
But now all breakthroughs are not 'picked' 
They are 'built' 
The roots of this tree are growing madly; 
It’s not that we cannot reach the fruit, 
We are using an action that has become obsolete, 
To understand a world that has already changed.
