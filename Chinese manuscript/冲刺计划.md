## 🎯 指数函数专属“5 分锦囊”

每天花 5 分钟默写以下内容：
```text
1. d/dx [e^x] = e^x
2. d/dx [a^x] = a^x ln a
3. d/dx [e^{u(x)}] = e^{u(x)} · u'(x)
4. ∫ e^{kx} dx = (1/k)e^{kx} + C
5. e^x = 1 + x + x²/2! + x³/3! + … （∀x）
6. 解 dy/dx = ky → y = Ce^{kx}
```
